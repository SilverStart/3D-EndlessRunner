# 3D-EndlessRunner
템플런이라는 게임이 생각나서 만든 엔들레스 러너 방식의 게임입니다. 동영상은 루트 폴더의 Movie_001.mp4 입니다.

# 사용한 기능
 - Character Controller 를 썼기 때문에 점프 기능 및 떨어지는 중력 현상을 직접 구현
 - List 와 반복문, Switch 문 과 Random 함수를 이용하여 바닥 타일과 떨어지는 공간 타일을 랜덤하게 생성하여 무한하게 달릴 수 있는 타일 생성 기능 구현
 - 획득 가능한 코인을 랜덤 위치에 계속 생성하고 획득하면 점수를 획득하며 점수가 3000점 씩 오를 때 마다 속도가 올라가는 기능 구현

# 추가하고 싶은 기능
 - 현재는 유저 속도가 오를 경우 그에 따른 맵과 코인 생성 로직이 유동적으로 변하지 않고 있기에 속도에 따라 맵을 더 자연스럽게 생성할 수 있도록 변경
 - 코인 말고도 부딪히면 체력이 깎이거나 게임 오버 되는 등 추가적인 오브젝트의 추가

# 어렵다고 느낀 부분
 - 3D 프로젝트를 하다보니 캐릭터 에셋의 애니메이션을 원하는 방식으로 수정하는 게 되게 어려웠습니다. 또 점프 로직을 구현할 때 실제 물리 관련 로직이 포함되어야 하다 보니 좀 복잡했지만 관련 이론을 Gemini 에게 여러번 물어보면서 했기에 어떻게든 이해하고 구현해볼 수 있던 것 같습니다.
